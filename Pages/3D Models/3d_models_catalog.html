Sword of Apris - Feather Damascus Longsword

This is a model of a longsword I made for a D&D campaign. I have dubbed it the Sword of Apris. It features feather damascus and a three-metal wire wrap on the hilt.

Modeled and rendered in blender using the Eevee rendering engine.

Texturing was done in Substance Painter.

Github Repo for the Sword of Apris

Gilded Longsword

This longsword is designed in the fashion of a ceremonial sword from the high medieval ages. Gilded with gold throughout and engraved with runes on the blade declaring the nobility of the wielder.

Modeled and rendered in blender using the Eevee and Cycles rendering engines.

Github Repo for the Gilded Sword model

Claymore - Steel and Copper

A model of a Claymore with a double fuller. A more rugged set of materials than the previous sword models, this one is made of a high carbon steel and features copper and iron fittings.

Modeled and rendered in blender using the Cycles rendering engine.

Github Repo for the Claymore model

Revolving Rifle

A firearm concept of my own creation. A break action revolving rifle chambered in 45-70 gov. Not sure that a break action system like this would actually work in real life with a round like 45-70, but in the world of 3D the sky is the limit.

Modeled and rendered in blender using the Eevee rendering engine.

Github Repo for the Revolving Rifle model
(no textures included)

Plasma Big Iron

Another concept of my own creation, a .44 magnum Big Iron.

"Made complete by a 23rd century conversion from conventional .44 mag shells to nuclear powered hypersonic plasma rounds. Conveniently facilitated by a simple conversion from the stock cylinder to the two part plasma cartridge cylinder and electronic firing pin manifold. The electronic firing pins allow for circuit completion within the plasma cartridges nuclear fuel cell upon hammer strike. The cell's thermal overload causes a slug of reactive material to begin melting as it is propelled down the barrel to over 1830 m/s and a maximum temperature of 10000℃.""

Modeled and rendered in blender using the Eevee rendering engine.

Github Repo for the Plasma Revolver model
(no textures included)

Bullpup Battle Rifle

This is one of my favorite firearm models I have created. Unfortunately I haven't been able to really finish it yet, so what you are seeing is a work in progress, but I think its far enough along to appreciate.

Chambered in 7.62x39 this bullpup rifle is the mechanical and stylistic mashup of something between an AK-47 and an IWI Tavor. The operating system is long-stroke gas piston, heavily inspired by the AK-47 and RPK platforms. The body of the firearm however takes its inspiration from the ergonomics and layout of the Tavor with a bit of AR-15 inspired M-LOK modular rail for good measure.

Modeled and rendered in blender using the Eevee rendering engine.

Github Repo for the Bullpup Battle Rifle project
(model state as shown, no textures included)

Emerald Birthstone Ring - Platinum Shank

One of the many designs I made while I was experimenting with casting rings.

A platinum band with an emerald set in one of two optional setting designs. I personally prefer the first one. The material for the Emerald was achieved by the use of a node set-up that allows for the photorealistic rendering of pretty much any gemstone imaginable.

Modeled and rendered in blender using the Cycles rendering engine.

(no textures included)
